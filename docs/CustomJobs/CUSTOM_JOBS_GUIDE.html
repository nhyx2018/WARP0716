Custom Jobs Guide

Everything you need to add custom jobs to a 2025-07-16 Ragnarok Online client using the CustomJobs WARP patch.

Requirements
WARP 07-16, rAthena, sprite editor (optional)
Job ID Range
4345 – MaxJob (default 4500)
Template
See the example/ folder

1 Choose a Job ID

Pick an unused ID between 4345 and your MaxJob setting (default 4500). The ID must match on both client and server.

Warning: IDs below 4345 conflict with existing rAthena jobs (Summoner, Doram, etc). IDs must not exceed your MaxJob value in the WARP profile.

For this guide, we'll use 4435 as our example ID and MY_CUSTOM_JOB as the name.

2 Client-Side Lua Files

All Lua files are in data/luafiles514/lua files/JobInfo/. Edit two files:

PCIds.lub — Job ID Definition

Add your job to the PCIds table (in the "Extended 2nd Classes" section):

PCIds.lub
MY_CUSTOM_JOB   = 4435,
MY_CUSTOM_JOB_F = 4436,  -- optional: female variant with different sprite

PCFuncs.lub — Job Properties

Add entries at the bottom of PCFuncs.lub:

PCFuncs.lub
-- Sprite folder name (MUST match your .spr/.act filename exactly, ASCII only)
PCPaths[PCIds.MY_CUSTOM_JOB]   = "MY_CUSTOM_JOB"
PCPaths[PCIds.MY_CUSTOM_JOB_F] = "MY_CUSTOM_JOB"

-- Display name (shown in BasicInfo, char select, party window)
PCNames[PCIds.MY_CUSTOM_JOB]   = "Custom Job"
PCNames[PCIds.MY_CUSTOM_JOB_F] = "Custom Job"

-- Inherit hand sprites, palettes, and IMF from an existing job
-- Common bases: PCIds.NOVICE, PCIds.SWORDMAN, PCIds.KNIGHT
local baseJob  = PCIds.NOVICE
local baseHand = PCHands[baseJob]
local basePal  = PCPals[baseJob]
local baseImf  = PCImfs[baseJob]

local myJobs = { PCIds.MY_CUSTOM_JOB, PCIds.MY_CUSTOM_JOB_F }
for _, jid in ipairs(myJobs) do
    if jid and not PCHands[jid] then PCHands[jid] = baseHand end
    if jid and not PCPals[jid]  then PCPals[jid]  = basePal  end
    if jid and not PCImfs[jid]  then PCImfs[jid]  = baseImf  end
end
Warning: Do NOT edit lub files with a text editor that corrupts CP949 (Korean) encoding. The existing entries contain Korean text. ASCII-only additions at the bottom of the file are safe with any editor. For edits to Korean content, use a hex editor or Python binary I/O.

Using Existing Sprites as Placeholders

If you don't have custom sprites yet, copy an existing job's sprites and rename them. The example/ folder includes Novice sprites you can use as a starting point.

4 IMF Files

IMF (Interval Motion Format) files control animation timing. Place them in data/imf/:

data/imf/MY_CUSTOM_JOB_남.imf    (male)
data/imf/MY_CUSTOM_JOB_여.imf    (female)

Copy from any existing job. The example/ folder includes Novice IMF files.

Critical: Without IMF files, your character will be invisible in-game (shadow only visible). This is the most commonly missed step.

5 Icon Files

Party UI icons go in data/texture/유저인터페이스/renewalparty/:

icon_jobs_4435.bmp          (normal icon, 24x24 BMP)
icon_jobs_4435_die.bmp      (dead icon, 24x24 BMP)

Replace 4435 with your actual job ID. Copy from any existing job icon as a placeholder. The example/ folder includes the Novice icon.

Warning: Missing icon files may cause crashes when viewing the party window with your custom job in the party.

6 WARP Profile Settings

In your WARP profile YAML, ensure CustomJobs is enabled and MaxJob covers your IDs:

profile.yml
patches:
  CustomJobs: on

$maxJob:
  display: 4500    # Must be >= your highest job ID + 1

Then rebuild the exe using WARP (GUI or console).

7 Server-Side (rAthena)

A. Job ID — src/common/mmo.hpp

mmo.hpp
// Add to the e_job enum:
JOB_MY_CUSTOM_JOB = 4435,

B. Map ID — src/map/map.hpp

map.hpp
// Add to e_mapid enum:
MAPID_MY_CUSTOM_JOB,

C. Job Conversion — src/map/pc.cpp

pc.cpp
// In pc_jobid2mapid():
case JOB_MY_CUSTOM_JOB:
    return MAPID_MY_CUSTOM_JOB;

// In pc_mapid2jobid():
case MAPID_MY_CUSTOM_JOB:
    return JOB_MY_CUSTOM_JOB;

D. Script Constants — src/map/script_constants.hpp

script_constants.hpp
export_constant(JOB_MY_CUSTOM_JOB);

E. Skill Tree — db/import/skill_tree.yml

skill_tree.yml
- Job: My_Custom_Job
  Inherit:
    - Job: Novice
  Tree:
    - Skill: SM_SWORD
      MaxLevel: 10
    # ... add your skills

F. Job Change NPC (example)

npc script
prontera,150,180,5	script	Job Master	2_M_SAGE,{
    if (BaseLevel < 10) {
        mes "Come back at base level 10.";
        close;
    }
    jobchange JOB_MY_CUSTOM_JOB;
    mes "You are now a Custom Job!";
    close;
}

Troubleshooting

"Cannot find File" error on login
Missing sprite or costume_1 files
Add all 8 sprite files (base + costume, both genders)
Character invisible (shadow only)
Missing IMF files
Copy IMF from an existing job and rename
Crash on party window
Missing icon BMP files
Add icon_jobs_ID.bmp and _die.bmp
Shows "Poring" as job name
PCNames entry missing in PCFuncs.lub
Add PCNames[PCIds.YOUR_JOB] = "Name"
Wrong sprite (shows Novice)
PCPaths doesn't match filename
Ensure PCPaths value matches sprite filename (case-sensitive)
Client crash on character select
Job ID exceeds MaxJob
Increase MaxJob in WARP profile YAML
"Cannot find File" during char creation
Outdated CustomJobs.qjs
Update to latest CustomJobs.qjs
